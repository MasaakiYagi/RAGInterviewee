# これはなに？

OpenAI の AssitantsAPI（RAG）を使った面談シミュレータ PoC

# コード以外に必要なものはある？

- OpenAI の API キー
- OpenAI の AssistantsAPI の Assistants_ID
- Python 環境
  - Version 3.12 系
  - Lib は requirements.txt
- ffmpeg
  - 公式からインストール[https://ffmpeg.org/download.html]
  - パス通す

# 実行手順

streamlit run streamlit_app.py
